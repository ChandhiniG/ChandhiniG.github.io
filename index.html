Some Things I've built

Below are some of my projects built as a hobby or as a part of my coursework

Parking availability prediction

Built a predictive model which gives the percentage of parking availability in every level of the building. Trained the model using Logistic regression and obtained an accuracy of 71%

Audio based activity recognition

Trained Random Forest Model on Raspberry pi to predict the activities done by a person in real time. Collected data using Microphone and extracted features like MFCC, Delta and DDelta and fed it to the model

  • Github

Visual Question Answering

Built a model which when given an image and a natural language question, predicts the top five most probable answers. Trained a CNN based on Resnet Architecture, LSTM and a classifier to produce the final answers.

  • Github

Predicting output of Tennis Match

Trained a model using Logistic Regression, Support vector machine and CNN to predict output of tennis match. Handcrafted the features from the dataset and obtained an accuracy of 73.56%

Stock Market prediction

Modeled a predictor which guesses the stock market price as a linear combination of previous three indices. Used linear regression to achieve this.

Six degrees of Kevin Bacon

Built a actor- movie connection from a dataset of 356000 points. Found the shortest path between the actors using BFS, DFS, Dijkstra and optimized it using disjoint set and A* algorithm. Verified the claim of "six degrees of kevin Bacon" with the dataset.
